# Data Processing Agreement (DPA)

If you are a customer based in the EEA or UK and use our Services to process personal data, we are required to enter into a Data Processing Agreement (DPA) with you. This DPA outlines our responsibilities as a data processor, including:

Instructions: We process personal data only on documented instructions from the data controller.
Confidentiality: We ensure that personnel authorized to process personal data are bound by confidentiality obligations.
Security: We implement appropriate technical and organizational measures to ensure the security of the personal data.
Sub-Processing: We only engage sub-processors with prior authorization from the data controller and under written contracts.
Data Breach Notification: We will notify the data controller without undue delay after becoming aware of a personal data breach.
Data Retention: We will return or delete personal data at the end of the contract unless required to retain it by law.
